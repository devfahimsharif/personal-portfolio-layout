Aniamted Calculator

The Animated Calculator is an interactive web-based tool that allows users to perform basic arithmetic operations (addition, subtraction, multiplication, division) with smooth visual animations for button clicks, number transitions, and results display.Built with HTML5, CSS3 animations/transitions, and JavaScript, this project enhances the traditional calculator experience by adding engaging UI effects such as button hover glow, result fade-in, and sliding number animations.The main goal of this project is to practice DOM manipulation, event handling, and CSS animations, while creating a user-friendly interface that feels modern and responsive.
